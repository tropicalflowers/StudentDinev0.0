📞 TROUBLESHOOTING:

Q: Landing page doesn't load?
A: Make sure server is running: python -m http.server 8000

Q: Login modal still appears?
A: Try clearing localStorage: Press F12 → Application → Clear All

Q: Theme not persisting?
A: Enable localStorage in browser settings

Q: Switch Role link not working?
A: Use local server instead of direct file access

Q: "403 Forbidden" error?
A: Check file permissions, run from project directory
